**Round 2 D-019 4Q** (用来锁 D-085 §2.2 ~ §2.5)

**Q5 — 默认 mode (启动后进哪个)**

- **(a)** Quiz mode（做题驱动，高频真题；偏 "备考" 任务）
- **(b)** Study mode（顺序阅读 trilingual，章节驱动；偏 "学习" 任务）
- **(c)** Chat mode（直接问 AI，AI 持 v1.0.3 全书 context；偏 "答疑" 任务）
- **(d)** Resume last（无固定默认，记住上次位置）
- **(e)** First-launch picker + remembered（首次启动选一次，之后记住）

**Q6 — Mode 切换机制**

- **(a)** Top tabs 3 水平 tab（Quiz / Study / Chat，SaaS 常见 pattern）
- **(b)** Hotkey 切换（如 Cmd+1/2/3）+ 可见 mode 指示
- **(c)** Side panel hybrid（Quiz/Study 主区 + Chat 常驻右侧 panel / drawer）
- **(d)** Progressive surface（单一 canvas，mode contextual — 例如点击 term 直接打开 Chat about it）
- **(e)** 其他 / 组合

**Q7 — Chat scope**

- **(a)** Whole book（AI 每次 session 有 v1.0.3 全 554 页 corpus 作 context）
- **(b)** Current chapter（AI scope 限当前章节）
- **(c)** Currently-viewing entity（最窄 — 只当前 term / question / page）
- **(d)** User-adjustable（默认 chapter，可手动展开到 whole book）
- **(e)** Mode-dependent：Study 模式下 = current chapter / Quiz 模式下 = current question / 独立 Chat tab = whole book

注：Whole book scope ≈ 数百 K~M tokens 输入（v1.0.3 zip 是 1.8 MB compressed JSON），LLM cost / latency 会成预算关键变量。本题答案直接影响 Session 27 step 4 LLM 模型选择。

**Q8 — Quiz/Study ↔ Chat 交互点**

- **(a)** 零集成（只有独立 Chat tab，user 手动复制问 AI）
- **(b)** Quiz "Explain" 按钮：错题 / 卡题时一键开 Chat，自动带问题 context
- **(c)** (b) + Study term hover/click → quick popover with kana_helper + "ask more" → Chat
- **(d)** (c) + Chat-initiated jumps: Chat 可建议 "study this chapter" / "try these 5 questions"，触发 Study/Quiz 跳转
- **(e)** (d) + 每个 entity 持久化 Chat 线程（Chat history per term / per question）
- **(f)** (e) + 间隔复习（AI 跟踪 mastery + 触发复习提示）

注：复杂度阶梯。Phase 2 α-now 自用阶段可以从 (b)/(c) 起步；(e)/(f) 是 β/γ 才有 ROI 的功能。Q8 答案 = D-085 §2.5 lock + Phase 2 α 阶段实施范围。
