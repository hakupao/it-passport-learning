- 输入：40 pages structured/ + glossary/glossary.json (55 entries)
- 每页 batch 1 LLM call → 估 ~30 calls (有些页全 glossary 命中跳过)

- Wall-time 估 ~15-20 min
- 累计 shadow 推进到 ~$12（hard cap $30 仍有 $18 余量）

- TranslationEngine 在调 LLM 前先用 glossary `by_jp_surface()` resolve standalone surface → 不进 LLM

- skip_existing 默认 ON，可 idempotent 重跑
