## Item 3 — 図表 96 件の修復

### 方式：主ループ編成 + workflow 並列ビジョン（loop-until-dry）

```
ESTIMATE (workflow 並列, general-purpose) — 源ページ PNG を視覚読取 → bbox 再推定 + is_figure 独立判定
  ↓ 主ループ：程序的余白 + PIL で確定的裁剪 → staging
VERIFY (workflow 並列, explore ≠ estimator → Rule D) — 新切り出し vs 源ページ → PASS/FAIL + 具体修正指示
```
