## 1. 用户反思 & 方向转换

用户核心洞察：

> 书之所以为书，有着他对于空间的设计以及排版，不适合一比一提取复刻，效果肯定不如原版书。

结论：放弃从教科書提取内容的路线（Stage 8-11），转向使用 IPA 官方源作为知识基础，用 AI 生成全新的三语教科书。

### D-019 两轮问答

## 3. Decisions

### D-108 LOCKED — Phase 5 方向: 基于 IPA 官方源的 AI 教科書生成

See `docs/decisions/D-108-phase5-ipa-source-ai-textbook.md`.

Major:
- 放弃书提取路线（Stage 8-11 中止）
- 新数据源: IPA 官方シラバス + 過去問 + 試験要綱 + IT用語集
- AI 生成三语教科書（日主 + 中 + 英）
- 保留现有 Web App 基础设施，换数据源
- LLM API 成本门槛取消（CLAUDE.md 已更新）

### D-109 LOCKED — Phase 5 数据目录结构

- `data/ip/sources/` — 下载的 IPA 原始 PDF
- `data/ip/syllabus/` — シラバス提取 JSON
- `data/ip/exams/` — 過去問提取 JSON
- `data/ip/textbook/` — AI 生成的教科書内容
- 简化目录结构，不再使用 `runs/<run_id>/` 层级（Phase 1 的 run-based 结构不适用于新方向）
