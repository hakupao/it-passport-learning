### 2. 全量ページマッピング — 二重チャネル方式

**Pass 1: Tesseract OCR** (`scripts/build-page-mapping.py`)
- 29 套 × ~50 ページ = 1,512 ページを Tesseract 5.5.2 (jpn) で OCR
- 正規表現 `[問間]\s*(\d{1,3})` + スライディングウィンドウフィルタで題号検出
- 結果: 2,689 題直接検出 + 211 題補間 = 2,900 題

**Pass 2: Claude Vision Agent** (29 並行 agent, Sonnet)
- 各試験 1 agent、8 ページ/バッチで全ページ読取
- 題号 + 図表有無 + 図表 bbox (分数座標) を出力
- 5 バッチ × 6 agent 並行 = 29 套完了

**比対結果** (`scripts/compare-and-merge-mappings.mjs`):
- 一致率: **95.8%** (2,779/2,900)
- 不一致: 121 題、**全て delta=1**（1ページ差）
- 抽検で Vision が全勝 → Vision を最終採用値として merge

### 3. 図表裁剪 + JSON 回写

**裁剪** (`scripts/crop-and-update.mjs`):
- Vision の fig_bbox 座標で Python/PIL 裁剪
- 502 枚裁剪 → `data/ip/exams/figures/` に保存
- `question_bank.json` + 29 個 `by_year/*.json` に `source` / `figure_path` / `figure_bbox_pct` / `figure_type` 回写

**全量検証** (13 バッチ × agent):
- 502 枚全数を目視検証
- **PASS: 400 / FAIL: 102 (20.3%)**
- FAIL 原因: bbox 不正確による截断（上部/下部/右側）、誤標記（図無し題に図フラグ）、空白画像 1 枚
- FAIL 一覧: `data/ip/exams/figures/_all_fails.json`
