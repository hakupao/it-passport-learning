## [JP]

### 絶対参照

複写の際にセル番地を固定する参照方法。列名または行番号の前に「$」を入力すると、複写時に値を固定できる。列のみを固定する場合（$B2）、行のみを固定する場合（B$2）、列と行の両方を固定する場合（$B$2）がある。

### 複写

セルの値をコピーすること。

## [ZH]

### 绝对引用

复制时固定单元格地址的引用方式。在列名或行号前输入「$」，即可在复制时固定该值。包括仅固定列的情况（$B2）、仅固定行的情况（B$2）以及同时固定列和行的情况（$B$2）。

### 复制

## [EN]

### Absolute Reference

A reference method that fixes the cell address when copying. By entering "$" before the column name or row number, the value can be fixed during copying. There are cases of fixing only the column ($B2), fixing only the row (B$2), and fixing both the column and row ($B$2).

### Copy

To copy the value of a cell.
