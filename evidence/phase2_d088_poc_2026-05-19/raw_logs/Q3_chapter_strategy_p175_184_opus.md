# 第4章 e-ビジネス・IoT・组込みシステム 章节速记（p175–184）

- **物联网（IoT）三要素架构** — `IoT（アイオーティー / aiōtī）`：汽车、家电等"物"直接接入互联网并互相交换信息。考点结构固定为「IoTサーバ（sāba，服务器，接收数据并下发指令）+ IoTデバイス（debaisu，设备）+ IoTゲートウェイ（gētowei，网关，替代无联网能力的设备中继通信、减轻网络负载）」，电力公司远程抄表是教科书级 IoT 实例（令和2年度真题）。

- **传感器与执行器的角色对立（极高频考点）** — `センサー（sensā，传感器）` 与 `アクチュエーター（akuchuētā，执行器）`：センサー把温度/湿度/光/声等物理量「检测并转为电信号」（输入侧），アクチュエーター把电信号「转为机械运动」来控制设备（输出侧）。令和2年度真题"开关窗户的电机"必选执行器，切勿与"接收数据/发送指令"等服务器职责混淆。

- **嵌入式系统与固件** — `組込みシステム（kumikomi shisutemu，嵌入式系统）`：为「特定功能」而内置于民生機器（电饭煲、空调、数码相机自动对焦）或産業機器（工业机器人、自动售货机）的微型计算机；其写入 ROM 的程序称 `組込みソフトウェア / ファームウェア（firmware，固件）`。题干关键词"特定の機能"基本一锤定音选 C（区别于专家系统/企业系统/单工系统）。

- **IoT 应用场景与社会愿景** — `Society 5.0（ソサエティ5.0）` 与 `インダストリー4.0（indasutorī 4.0，工业4.0）`：Society 5.0 是融合网络空间与物理空间、以人为中心的日本国家愿景；工业4.0 是用 IoT+AI 革新制造业（继蒸汽机、电力、计算机后的第四次产业革命）。配套术语 `コネクテッドカー（konekuteddo kā，互联汽车，令和3年度真题）`、`スマートファクトリー（智能工厂）`、`ドローン（dorōn，无人机）` 常作选项区分。

- **e-Business 营销模式辨析** — `レコメンデーション（rekomendēshon，推荐）` 与 `フリーミアム（furīmiamu，免费增值）`：推荐＝分析浏览/购买历史向个人推送适配商品（区别于アフィリエイト联盟营销、フラッシュマーケティング闪购）；フリーミアム＝「基本功能免费、附加功能收费」的商业模式（名片管理服务的"个人登记免费+企业共享收费"是标准案例，区别于订阅制和广告免费刊物）。配套术语还有 `CGM（消费者生成媒体）`、`エスクローサービス（esukurō sābisu，第三方担保）`。
