# 第 181 页 · 第 1 题 ｜ 网购营销手法辨析

## 题目回顾

> 在网络购物中，分析个人访问过的网页**浏览历史**和**商品购买历史**，向用户**显示可能感兴趣的信息以促进购买其他商品**的营销手法是哪一项？

这道题的核心信号词是 **「閲覧履歴 / 購入履歴 → 関心のありそうな情報 → 別商品の購入を促す」**（浏览/购买历史 → 推测兴趣 → 推动买别的东西）。看到这串信号，就要立刻锁定「基于个人行为数据做个性化推送」的概念。

下面把 4 个片假名选项逐一拆开，给中文学习者做发音 + 概念 + 判断三件套。

---

## ア．アフィリエイト

- **片假名 → 罗马字**：`アフィリエイト` = **a-fi-ri-e-i-to**（拼读时连读成「阿菲利耶伊托」，对应英文 *affiliate*）。
  - 拆音节：ア(a) / フィ(fi) / リ(ri) / エ(e) / イ(i) / ト(to)。注意「フィ」是拗音，发 *fi*，不是 *hu* + *i*。
- **中文概念**：**联盟营销 / 联属营销**。商家在第三方网站（博主、媒体、个人）上放置专属链接或广告，访客通过该链接产生购买/注册后，第三方按成交分润获得佣金。
- **是否正确**：❌ 不对。アフィリエイト 关注的是「**广告投放渠道 + 按效果分成**」的商业模式，主语是「商家 ↔ 站外合作伙伴」，并不涉及对**单个用户行为历史**的分析与个性化展示。题目讲的是 EC 站**内部**针对**单个登录用户**的个人化推送，与联盟营销不是一回事。

---

## イ．オークション

- **片假名 → 罗马字**：`オークション` = **ō-ku-shon**（长音 ō；近似「欧——库雄」，对应英文 *auction*）。
  - 拆音节：オ(o) / ー(长音，把前面的 o 拉长) / ク(ku) / ション(shon)。
- **中文概念**：**拍卖 / 竞价交易**。多名买家在限定时间内出价竞争，由出价最高者成交（如雅虎拍卖 Yahoo!オークション、eBay）。
- **是否正确**：❌ 不对。オークション 是一种**价格发现机制**，靠多方竞价决定成交价，和「分析浏览/购买历史推荐别的商品」毫无关系。本题谈推送，不谈定价。

---

## ウ．フラッシュマーケティング

- **片假名 → 罗马字**：`フラッシュマーケティング` = **furasshu māketingu**（「夫拉嘘 马——开听古」，对应英文 *flash marketing*）。
  - 拆音节：フ(fu) / ラ(ra) / ッ(促音，下一个辅音翻倍，写作 ssh) / シュ(shu) / マ(ma) / ー(长音 ā) / ケ(ke) / ティ(ti) / ン(n) / グ(gu)。
- **中文概念**：**闪购营销 / 限时抢购促销**。在**极短时间窗**内（数小时到几天）以**大折扣 + 最低成团人数**的形式刺激集中下单，典型如 Groupon、团购日特价。
- **是否正确**：❌ 不对。フラッシュマーケティング 的关键是「**时间紧迫 + 集体折扣**」，靠稀缺感推动转化，并不依赖**个人历史数据**做个性化。题目里完全没有「限时」「折扣」「团购」的信号词。

---

## エ．レコメンデーション ✅

- **片假名 → 罗马字**：`レコメンデーション` = **rekomendēshon**（「雷靠门戴——雄」，对应英文 *recommendation*）。
  - 拆音节：レ(re) / コ(ko) / メ(me) / ン(n) / デ(de) / ー(长音 ē) / ション(shon)。注意「デー」是长音，要把 e 拉长，别读成短促的 de。
- **中文概念**：**推荐（系统）/ 个性化推荐**。通过分析**单个用户**的浏览、点击、购买、收藏等行为日志，结合协同过滤、内容相似度等算法，主动展示「你可能也喜欢」「买了 X 的人还买了 Y」类信息，引导其购买其它商品。亚马逊的「おすすめ商品」是教科书级范例。
- **是否正确**：✅ **正确答案 = エ レコメンデーション (Recommendation)**。
  - 题干三个关键词与定义**逐字对应**：
    1. 「閲覧履歴・購入履歴を**分析**」→ 推荐系统的数据来源。
    2. 「**関心のありそうな**情報を表示」→ 推荐系统的个性化输出。
    3. 「**別商品の購入を促す**」→ 推荐系统的交叉销售（cross-sell）目标。
  - 其余三个选项各自缺少「基于个人行为数据做个性化推送」这一核心特征，因此被淘汰。
